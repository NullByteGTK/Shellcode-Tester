# Shellcode Tester
A simple C program for testing shellcode

Usage:
  $ ./shellcode
  Shellcode: \<yourshellcode\>
  Shellcode address: \<Your shellcode's address on heap\> - Length: \<Your shellcode's length\>
  \<Runs your shellcode\>
