# Shellcode Tester
A simple C program for testing shellcode

Usage:<br>
  $ ./shellcode<br>
  Shellcode: \<yourshellcode\><br>
  Shellcode address: \<Your shellcode's address on heap\> - Length: \<Your shellcode's length\><br>
  \<Runs your shellcode\>
